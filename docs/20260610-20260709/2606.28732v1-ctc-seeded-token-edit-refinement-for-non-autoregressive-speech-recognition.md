## 摘要
非自回归自动语音识别（ASR）支持并行解码，但许多基于精炼的方法从随机、全掩码或固定长度的令牌序列开始，需要多次迭代才能重建完整转录。我们将ASR解码形式化为对贪婪连接时序分类（CTC）假设的变长编辑精炼。一个声学条件编辑流解码器直接在压缩的CTC假设上操作，并行预测插入、删除和替换操作。编辑流解码器与CTC模型通过连续时间离散扩散损失进行联合训练。在推理阶段，我们发现仅需两个编辑步骤就能大幅降低词错误率（WER），而无分类器引导（CFG）通过让模型聚焦于音频特征进一步增强了识别质量。我们还利用CTC置信度约束编辑提议以提高准确性。最后，消融研究验证了我们的设计选择，而解码器预训练和预训练编码器集成带来了显著的额外性能提升。

## Abstract
Non-autoregressive automatic speech recognition (ASR) enables parallel decoding, but many refinement-based methods begin from random, fully masked, or fixed-length token sequences, requiring multiple iterations to reconstruct the complete transcript. We instead formulate ASR decoding as a variable-length edit refinement of a greedy connectionist temporal classification (CTC) hypothesis. An acoustic-conditioned Edit Flow decoder operates directly on the collapsed CTC hypothesis, predicting insertion, deletion, and substitution operations in parallel. The Edit Flow decoder is jointly trained with a CTC model using a continuous-time discrete diffusion loss. During inference, we find that just two edit steps yield substantial Word Error Rate (WER) reductions, and classifier-free guidance (CFG) further enhances recognition quality by focusing the model on audio features. We also constrain edit proposals using CTC confidence to improve accuracy. Finally, ablation studies validate our design choices, while decoder pretraining and pretrained encoder integration yield significant additional performance gains.

## 2. 方法论

### 2.1 核心思想
- 将ASR任务分解为：**CTC首遍产生短假设** → **声学条件编辑流解码器纠正残余错误**。
- 解码器直接在压缩（去除空白符）的CTC令牌序列上操作，并行预测**插入、删除、替换**三种编辑操作，无需固定长度或空白填充。

### 2.2 关键技术细节
- **首遍CTC**：给定声学特征 \(A\)，编码器输出帧级后验概率 \(p_{\text{ctc}}(v|h_t)\)。贪婪对齐经去除重复字符与空白符后得到初始令牌序列 \(\hat{Y}^{(0)}\)。
- **编辑流扩散模型**：将 \(\hat{Y}^{(0)}\) 与真值 \(Y\) 对齐到含间隙符 \(\epsilon\) 的辅助空间，定义连续时间噪声调度 \(\kappa(t)\)，通过随机掩码构建中间状态 \(z_t\)。移除间隙后得到当前序列 \(x_t\)。
- **声学条件编辑场**：双向Transformer解码器输入 \(x_t\)、流时间 \(t\) 和声学记忆 \(H\)，预测每个位置的动作速率 \(\lambda^{\text{ins}}, \lambda^{\text{sub}}, \lambda^{\text{del}}\) 及令牌分布 \(q^{\text{ins}}, q^{\text{sub}}\)。
- **训练目标**：使用连续时间离散扩散损失 \(L_{\text{EF}}\) 与CTC损失联合训练：\(L = \alpha L_{\text{CTC}} + (1-\alpha)L_{\text{EF}}\)，\(\alpha=0.3\)。
- **文本预训练**：仅用文本构造编辑噪声（删除、替换、插入）对 \(( \tilde{Y}, Y )\)，用同一编辑流损失预训练解码器（声学记忆置零），提升初始纠错能力。

### 2.3 推理策略
- **多步迭代精炼**：使用泊松 τ-leaping 近似，按步长 \(h_k\) 计算插入和删除/替换的事件概率，超过阈值（0.1）则执行最高分编辑。默认 \(K=2\) 步（每步 \(h_k=0.5\)）。
- **音频无分类器引导（CFG）**：训练时以0.1概率丢弃声学记忆；推理时组合有/无条件速率和分布，通过引导尺度 \(w\) 加强声学条件依赖。
- **CTC置信度引导**：仅在CTC置信度低于阈值（0.7）的位置允许编辑。首步用贪婪对齐的置信度，后续步骤用当前假设的强制对齐置信度。

## 6. 主要结论与发现

- 从CTC假设出发的编辑精炼**仅需2步并行编辑**即可带来显著WER下降，远超CTC基线。

## 7. 优点

- **创新范式**：将NAR解码重塑为编辑纠正，避免从噪声/掩码生成全新序列，大幅减少所需迭代步数。
- **高效率**：序列长度仅为压缩后假设，远短于帧级对齐，且编辑操作直接改变令牌序列，无需独立长度预测器。
- **推理策略巧妙**：音频CFG确保声学条件依从，CTC置信度门控防止误编辑正确部分，二者协同提升鲁棒性。
- **预训练设计**：纯文本编辑噪声预训练简单有效，为解码器提供良好初始纠错能力。
- **模块化与可扩展性**：可搭配不同编码器（ESPnet、Whisper），增加编码器规模即可持续提升性能。
